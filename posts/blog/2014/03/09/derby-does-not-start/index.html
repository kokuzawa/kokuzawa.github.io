Derby does not start

March 9, 2014
Java Derby

Java8を入れたらGlassFishに付属しているDerbyが起動できなくなっていたので、下記を$JAVA_HONE/jre/lib/security/java.policyに追記しました。

permission java.net.SocketPermission "localhost:1527", "listen,resolve";
